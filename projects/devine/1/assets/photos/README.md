# Your photos go here

Drop your photos into this folder (e.g. `01.jpg`, `02.jpg`, …), then open
`../../main.js` and set each moment's `img` to its path:

```js
{ date: "Our first date", text: "...", img: "assets/photos/02.jpg" },
```

Tips:

- Portrait/vertical photos look best (the story is full-screen on a phone).
- Leave `img: ""` on any moment to keep the placeholder for now.
